Step 1: Create the project (not required for viewers)
On Vscode terminal type the following:
-npx create-next-app@latest ./

After that you will have to choose a Yes or No for a few options and the options below show exactly what you should choose:
TypeScript: Yes
ESLint: Yes
Tailwind.CSS: Yes
'src/' directory: Yes
App Router: Yes
import alias: No

Step 2: Installing the packages
To install the packages: npm install --save-dev framer-motion three react-intersection-observer @react-three/drei @react-three/fiber
npm install @heroicons/react
npm install react-icons

Step 3: Running the server
npm run dev
Then click on the link to open the server
